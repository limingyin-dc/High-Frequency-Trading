# C++ 关键字说明

本文档列出项目中实际用到的 C++ 关键字及其作用。

---

## 类型关键字

| 关键字 | 作用 |
|--------|------|
| `int` | 32位整型，用于计数、索引、状态码等 |
| `double` | 64位浮点数，用于价格、保证金、盈亏等金融数值 |
| `char` | 单字节字符，用于合约代码、方向标志、固定长度字符串 |
| `bool` | 布尔类型，用于标志位（如 `isReady`）和函数返回值 |
| `void` | 无返回值，用于不需要返回结果的函数 |
| `auto` | 自动推导类型，用于简化迭代器、lambda、复杂类型的声明 |
| `nullptr` | 空指针常量，替代 `NULL`，类型安全 |

---

## 类型修饰关键字

| 关键字 | 作用 |
|--------|------|
| `const` | 声明不可修改的变量或指针，防止意外写入，也用于成员函数表示不修改对象状态 |
| `static` | 修饰成员变量时表示类级别共享；修饰局部变量时生命周期延长到程序结束；修饰全局函数/变量时限制链接作用域 |
| `volatile` | 告知编译器不要优化该变量的读写，用于 `WarmUp()` 中防止预热循环被优化掉 |
| `constexpr` | 编译期常量，用于 `MAX_ORDERS`、`TICK_POOL_SIZE`、`SCALE` 等，零运行时开销 |
| `explicit` | 禁止构造函数的隐式类型转换，如 `TdEngine(const RiskConfig&)` 防止意外构造 |
| `inline` | 建议编译器内联展开函数，用于 `Tsc::Now()` 等热路径小函数，消除函数调用开销 |

---

## 存储与生命周期关键字

| 关键字 | 作用 |
|--------|------|
| `static` | （见上）同时控制存储期和链接性 |
| `alignas` | 指定变量或结构体的内存对齐字节数，`alignas(64)` 让结构体独占一条 cache line，避免 false sharing |

---

## 面向对象关键字

| 关键字 | 作用 |
|--------|------|
| `class` | 定义类，默认成员为 `private` |
| `struct` | 定义结构体，默认成员为 `public`，用于纯数据聚合（如 `SlimTick`、`OrderSlot`、`RiskConfig`） |
| `public` | 成员对外可见，接口函数和公开字段使用 |
| `private` | 成员仅类内可见，内部实现细节使用 |
| `namespace` | 命名空间，用于 `OrderState`、`PriceUtil`、`Tsc` 等，避免全局命名冲突 |
| `override` | 显式标注重写基类虚函数，编译器会验证签名匹配，用于所有 CTP SPI 回调 |
| `virtual` | 声明虚函数，CTP SPI 基类用此实现多态回调分发 |
| `this` | 指向当前对象的指针，用于 lambda 捕获（`[this]`）和成员访问 |
| `explicit` | （见上） |

---

## 内存管理关键字

| 关键字 | 作用 |
|--------|------|
| `new` | 堆上动态分配对象，CTP API 内部使用 |
| `delete` | 释放 `new` 分配的内存 |
| `sizeof` | 编译期获取类型或变量的字节大小，用于 `snprintf`、`strncpy` 的缓冲区长度参数 |

---

## 控制流关键字

| 关键字 | 作用 |
|--------|------|
| `if` / `else` | 条件分支，风控检查、状态判断等大量使用 |
| `while` | 循环，策略主循环 `while(true)`、风控滑动窗口弹出旧记录等 |
| `for` | 计数循环，遍历槽位数组、合约列表等 |
| `switch` / `case` / `default` / `break` | 多分支选择，`OnOrderUpdate` 中根据 CTP 订单状态分发处理逻辑 |
| `continue` | 跳过本次循环剩余逻辑，策略循环中无新 tick 时直接 `continue` |
| `return` | 函数返回，风控拦截时提前 `return ""` |
| `do` | `do-while` 循环，`UpdatePnl` 中用 CAS 循环原子更新 double |

---

## 类型转换关键字

| 关键字 | 作用 |
|--------|------|
| `static_cast` | 编译期安全的显式类型转换，如 `int` → `double`、`double` → `int64_t` |
| `reinterpret_cast` | 按位重新解释指针类型，CTP API 要求 `char*` 时使用 |

---

## 模板关键字

| 关键字 | 作用 |
|--------|------|
| `template` | 定义函数或类模板，`ForEachTimeout<Fn>` 用模板接受任意可调用对象作为回调 |
| `typename` | 在模板参数列表中声明类型参数 |

---

## 属性与标注

| 关键字 / 属性 | 作用 |
|--------|------|
| `[[nodiscard]]` | 标注函数返回值不应被忽略，`AllocSlot` 用此防止调用方丢弃返回的槽位指针导致槽位泄漏 |

---

## 其他关键字

| 关键字 | 作用 |
|--------|------|
| `true` / `false` | 布尔字面量 |
| `using` | 引入命名空间成员或定义类型别名，简化代码 |
| `operator` | 重载运算符，标准库容器内部使用 |
